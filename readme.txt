程序运行方法
1.运行前先输入“python3 -m pip install -r whl_is_sb.txt”或者“pip3 install -r whl_is_sb.txt”
2.code文件夹里面有个main.py，输入“python3 <路径>/code/main.py”或者“python <路径>\\code\\main.py”
3.然后请耐心地等待1个小时左右
4.运行结束之后，你会看到主文件夹里多了个test.mid文件，这个文件就是生成的文件。


说明：
1.共享噪音（Noise Maker） 0.92是我自制的一个简单的音（zao）乐（yin）生成程序。主要的生成算法就是深度学习deep learning。目前的这个版本可以生成一个带有主旋律 和弦 以及鼓点的简单音乐。这个程序生成的音乐确实不怎么好听，但是比上个版本要好一些，这次应该是能听出一些旋律的。
2.关于这些文件：code文件夹里面是源代码，data文件夹里面是训练用的midi和sqlite3数据。algorithm.docx是我简单总结了一下我这个程序的算法。“试听曲”文件夹中有三个试听曲。update.odt是版本更新日志。code_old文件夹是老版本代码的备份，没啥卵用。
3.我把使用到的算法挑重点写在了algorithm.docx中。但是这个版本的音乐生成方法比上个版本要复杂得多，而且涉及一些我也不太懂的的音乐术语，再加上我自己可能时间也不太够，所以在那上面写的算法就非常地简略，不太容易读懂（评论：你他妈就是偷懒能别找借口吗）。如果对我这个程序生成音乐的算法比较有兴趣的话，可以下来私聊我。
4.关于数据：我从网上找了169首儿歌作为训练样本。个人对儿歌比较感兴趣，再加上儿歌的midi文件比较好下载到，所以这次训练样本就选了儿歌。
5.关于这个程序的源代码：这个程序的源代码目前还比较长且凌乱，可读性比较差。过几天之后我应该会把代码整理好发到GITHUB上，对我这个程序的源代码感兴趣的话，可以关注我的GITHUB地址https://github.com/HaloOrangeWang。
